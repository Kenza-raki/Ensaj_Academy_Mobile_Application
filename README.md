# Ensaj_Universe_Mobile_Application

" Ensaj Academy " is a mobile application dedicated to simplifying academic management while enhancing the student experience. Teachers benefit from features allowing them to publish educational content, organize assessments, and track student progress. On the other hand, students have access to various tools enabling them to easily view their schedules, access course materials, and check their assessment results. With a user-friendly and intuitive interface, " Ensaj Academy " provides an optimal user experience, thereby enhancing student engagement and motivation in their academic journey


### Teacher Features
- **Meetings Planning**: Schedule meetings and view them in a shared calendar. Receive notifications for upcoming meetings.
- **Rooms Reservation**: Reserve specific rooms for activities via an integrated calendar, ensuring efficient use of school facilities.
- **Create QR Code**: Generate unique QR codes for each class session, allowing easy attendance tracking and verification.
- **Add Courses**: Seamlessly add new courses with detailed descriptions, materials, and resources for students to access.
- **Submit Results**: Manage student assessments, submit grades, and provide feedback efficiently through the app.

### Student Features
- **QR Code Attendance**: Scan QR codes to mark attendance during classes and academic events, simplifying attendance tracking.
- **View Courses**: Access a comprehensive list of enrolled courses with detailed information about course schedules and instructors.
- **View Schedule**: Check and plan academic activities with a personalized schedule that includes class timings and important events.
- **Check Results**: View academic results and performance metrics, including grades, modules completed, and upcoming assessments.
- **Take Quizzes**: Participate in interactive quizzes created by teachers to assess understanding and knowledge retention.

### Common Features
- **Authentication**: Secure login using email and password, ensuring data privacy and user authentication.
- **Profile Management**: Update personal information, preferences, and contact details conveniently within the app.
- **Interactive Communication**: Facilitate transparent communication between teachers and students through messaging and announcement features.

## Technologies Used

Ensaj Academy leverages modern technologies to deliver a robust and scalable mobile application:

- **Frontend**:
  - Dart
  - Flutter for cross-platform mobile development

- **Backend & Cloud Services**:
  - Firebase Authentication for user authentication
  - Cloud Firestore for real-time database management
  - Firebase Cloud Functions for server-side logic
  - Firebase Cloud Messaging for notifications

- **Additional Services & Integrations**:
  - Google APIs for Google Calendar integration to manage meetings and events
  - QR Code generation and scanning libraries for efficient attendance tracking
  - Integration with Google Meet for virtual meetings and classes
